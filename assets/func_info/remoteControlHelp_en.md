# Remote Control Help
How to control the remote computer

## [mouse,blue] Touchpad Area
- Move your finger anywhere on the touchpad area to move the mouse cursor
- Tap the touchpad area for a left click
- Long press the touchpad area to toggle left long-click mode
- Two-finger tap: Right click
- Two-finger scroll: Scroll the page
- Two-finger drag: Drag and drop


## [mouse,green] Mouse Buttons
- Use the Left/Right/Middle Click buttons for specific mouse clicks
- Long press Left/Middle Click buttons to toggle long-click mode (buttons will highlight when active)
- Long press is useful for text selection and drag operations

## [keyboard,orange] Keyboard Mode
- Switch to keyboard mode using the keyboard icon in the top bar
- Full 87-key TKL layout with all standard keys
- Press and hold keys for continuous input
- Supports up to 4 simultaneous key presses

## [text_fields,purple] Send Text
- Use the text icon in the top bar to send text messages
- Text will be typed directly at cursor position
- If direct typing fails, text will be pasted from clipboard
- Draft text is automatically saved when closing the dialog

## [settings,red] Other Controls
- Press Back button to disconnect (with confirmation)
